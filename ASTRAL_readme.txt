ASTRAL Readme

Files:
ASTRALspecies_pp.tre shows the resulting multispecies coalescent topology generated by ASTRAL from 984 genes. Support values are local posterior probabilities, and branch lengths in coalescent units.

ASTRALspecies_boot.tre shows the results of 100 replicates of multilocus bootstrapping. The last line is the summary of all bootstraps.
